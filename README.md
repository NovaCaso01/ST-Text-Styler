# 텍스트 꾸미기 (ST-Text-Styler) - SillyTavern Extension

텍스트 관련 다양한 기능을 제공하는 SillyTavern 확장프로그램입니다.

## ✨ 기능

### 🔒 단어 가리기
- 특정 단어/문장을 가리기
- 🎨 **색상**으로 가리기 (직접 색상 선택)
- 😊 **이모지**로 가리기 (하트, 펭귄, 로봇 등)
- ✨ **아스키아트**로 가리기 (구름, 나비, 리본 등)
- 📝 여러 단어를 각각 다른 방식으로 가리기
- 🔘 온/오프 토글 기능

### 🎨 글꾸미기 (AI)
- 채팅 내용을 게시글용 HTML로 변환
- AI가 자동으로 스타일 적용
- 커뮤니티 호환 HTML 생성

## 📱 사용법

1. 입력창 옆 **마법봉 버튼(✨)** 클릭
2. **"텍스트 꾸미기"** 선택
3. 팝업에서:
   - **단어가리기 탭**: 단어/문장 가리기 설정
   - **글꾸미기 탭**: AI HTML 생성
